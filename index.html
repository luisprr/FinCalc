Conceptos de Mercado Bursátil y Bolsa de Valores

  • Mercado bursátil: compra y venta de valores emitidos en la bolsa.
  • Bolsa de valores: sistema organizado que promueve la inversión y facilita la liquidación.
  • Relación clave: Inversionistas ↔ Comisionistas ↔ Creadores de mercado.
  • Bolsa de Valores de Lima: regula listado de títulos y facilita liquidación de operaciones.

Determinantes de Tasa de Interés

r = RR + IP + DP + MP + LP + EP (+ RP opcional)

Interés Simple

S = C * (1 + i * t) | I = C * i * t

Si ingresas Días, se usa t = días/base. Si ingresas Tiempo t, se usa ese valor directamente.

Interés Compuesto

S = C * (1 + TN/m)^n

Sugerencia de clase: TN anual y capitalización diaria, entonces m ≈ 360.

Pago Diferido y Descuento por Pronto Pago

i_max = (S/C - 1) / t

Pago con Cuota Inicial (Tasa Implícita)

i = (S/C - 1) / t
